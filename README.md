Retail Basket Analysis & Cross-Selling Optimization

## Project Overview
End-to-end SQL analysis of 500,000+ retail transactions to identify
product purchase patterns and cross-selling opportunities,
visualized in a Power BI dashboard.

## Business Problem
A UK-based online retailer wants to understand:
- Which products are frequently bought together?
- When do customers buy (seasonality)?
- Which customer segments drive the most revenue?

## Tools Used
- SQL (SQLite via DBeaver)
- Power BI Desktop
- Git & GitHub
- VS Code
